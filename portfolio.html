About Me

I'm Alex Rivera, a front-end developer with a deep passion for CSS architecture and responsive design systems. I believe great CSS is invisible — it adapts gracefully to every device, respects user preferences, and never breaks.

My workflow is mobile-first and component-driven. I use CSS Grid for complex two-dimensional page layouts, Flexbox for one-dimensional component alignment, and CSS Custom Properties as the backbone of every design system I build.

Outside of code, I contribute to open-source CSS toolkits, write about advanced layout techniques, and help teams migrate legacy float-based layouts to modern Grid/Flexbox architectures.
